# Movie Finder API
The Movie Finder API is a Node.js backend application built with Express that interacts with the OMDb (Open Movie Database) API to search for movies and retrieve detailed information about specific movies.

This API acts as a proxy between a client application and the OMDb API, allowing users to search for movies by title and retrieve detailed movie information using an IMDb ID.

## Technologies Used

Node.js

Express

Axios

Dotenv

OMDb API
